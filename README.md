# ufs-weather-project
Can the UFS Short Range Weather (SRW) Application simulate a high-impact convective event — the August 10, 2020 derecho — in a way that is both scientifically meaningful and accessible to new users? 

# What does this project do?
This project will run and test a UFS Short Range Weather Application tutorial (https://ufs.epic.noaa.gov/srw-tutorial-august102020-derecho/)
